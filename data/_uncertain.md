# Icônes / scalings incertains

Ce fichier liste les scalings extraits d'un screenshot dont l'icône de stat
n'a pas pu être identifiée avec confiance (champ `"stat": "unknown"` dans le JSON).

Légende probable (à confirmer au fur et à mesure) :
- violet -> Magic
- orange / rouge -> Attack
- vert -> HP ou Shield (à désambiguïser selon le contexte du texte)
- bleu -> Mana

## À vérifier

_(vide pour l'instant — aucun screenshot traité)_

Format d'une entrée :

```
- **<hero>** / <active|passive nom> : "<rawText>" — icône observée : <couleur/forme>, contexte : <suggestion>
```
